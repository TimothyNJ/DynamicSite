# Zitadel Cloud Configuration for DynamicSite

## Your Instance Details
ZITADEL_INSTANCE_URL=https://[YOUR-INSTANCE].zitadel.cloud
ZITADEL_CLIENT_ID=[YOUR-CLIENT-ID]

## Local Development URLs
REDIRECT_URI=http://localhost:8081/auth/callback
POST_LOGOUT_URI=http://localhost:8081

## Notes
- Replace [YOUR-INSTANCE] with your actual Zitadel instance name
- Replace [YOUR-CLIENT-ID] with the Client ID from your application
- These values will be used in the auth service configuration
