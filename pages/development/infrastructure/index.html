Infrastructure

GitHub

Source repository. GitHub Actions CI/CD deploys to S3 on push to development and production branches. PAT stored in AWS Secrets Manager for Lambda and API operations.

github.com/TimothyNJ/DynamicSite

Zitadel

Identity provider. PKCE OAuth2 login for all environments. Client ID 339931209931003985, Project ID 339930261431031889. Service account key in Secrets Manager used by Lambda proxy for Management API access.

CDN for dynamicsite.io and www.dynamicsite.io. Origin: tnjdynamicsite S3 bucket. Viewer-request function dynamicsite-prod-auth-redirect provides HTTP Basic Auth site blocker and www-to-apex redirect.

Distribution E28F8S6UWTY5P2

AWS S3

Static site hosting. Three buckets: tnjdynamicsite (production), tnjdynamicsite-dev (development), tnjdynamicsite-sandbox (sandbox). All deployed via GitHub Actions with no-cache headers. Also stores security-config.json and version.json.

S3 Console (us-west-2)

AWS Lambda

API proxy between the client and Zitadel Management API. Authenticates via service account JWT, handles session management, font variable push, and security config updates. Deployed from lambda/zitadel-proxy/ on code changes.

dynamicsite-zitadel-proxy (Python 3.12)

AWS Secrets Manager

Stores six secrets: Zitadel service account key, GitHub PAT, Cloudflare API token, Migadu API key, SES SMTP credentials, and GitHub Actions deployer IAM credentials.

Secrets Manager (us-west-2)

AWS SES

Email sending service. SMTP credentials stored in Secrets Manager. Used for transactional email from the DynamicSite domain.

SES Console (us-west-2)

Cloudflare

DNS management for dynamicsite.io zone. CNAME records point each subdomain (dev, sandbox, www) to the corresponding CloudFront distribution. API token (claude-dns-admin) stored in Secrets Manager.

dash.cloudflare.com

Migadu

Email hosting provider for the dynamicsite.io domain. Handles inbound mail, mailboxes, and forwarding. API key stored in Secrets Manager.

admin.migadu.com

Webpack

Bundles js/main.js into dist/bundle.js with Babel transpilation and SCSS loading. Generates version.json with Pacific-time build timestamp used by update-notifier. BUILD_TIMESTAMP injected via DefinePlugin.
